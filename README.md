# My Flask App

## How to Run

1. Create a virtual environment (optional but recommended):

python3 -m venv venv
source venv/bin/activate # On Windows use venv\Scripts\activate


2. Install Flask:

pip install -r requirements.txt

3. Run the app:

python app.py


4. Open your browser and go to `http://127.0.0.1:5000/`
